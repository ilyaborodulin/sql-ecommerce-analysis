# SQL-проект: анализ продаж e-commerce

## Описание проекта
В проекте выполнен SQL-анализ транзакционных данных интернет-магазина.

Цель проекта — изучить ключевые бизнес-метрики, проанализировать продажи, клиентов, географию выручки и сезонность спроса.

## Датасет
В проекте используется e-commerce датасет со следующими основными полями:
- `InvoiceNo` — номер заказа
- `StockCode` — код товара
- `Description` — описание товара
- `Quantity` — количество товара
- `InvoiceDate` — дата и время заказа
- `UnitPrice` — цена за единицу товара
- `CustomerID` — идентификатор клиента
- `Country` — страна клиента

## Цель проекта
Цель проекта — с помощью SQL рассчитать основные бизнес-метрики и выявить базовые закономерности в продажах интернет-магазина.

## Что было сделано
В рамках проекта были выполнены следующие SQL-задачи:
- создана таблица и загружен CSV-файл в PostgreSQL
- рассчитана общая выручка без возвратов
- рассчитано количество заказов
- рассчитано количество клиентов
- рассчитан средний чек
- проанализирована выручка по месяцам
- выделены топ-10 клиентов по выручке
- выделены топ-10 стран по выручке
- выделены топ-10 товаров по количеству продаж
- использована оконная функция для расчёта накопительных трат клиентов

## Основные результаты

### Базовые метрики
- Общая выручка без возвратов: **10,644,560.42**
- Количество заказов: **20,728**
- Количество клиентов: **4,372**
- Средний чек: **513.54**

### Динамика выручки по месяцам
Анализ показал выраженную сезонность:
- пик выручки пришёлся на **ноябрь 2011** — **1,509,496.33**
- высокие значения также наблюдаются в **сентябре** и **октябре 2011**
- в конце года продажи существенно выше, чем в начале

### Топ клиентов
Наибольшую выручку приносят отдельные крупные клиенты.

Топ-3 клиента по сумме покупок:
- `14646` — **280,206.02**
- `18102` — **259,657.30**
- `17450` — **194,550.79**

Это показывает, что заметная часть выручки формируется ограниченным числом покупателей.

### Топ стран по выручке
Основная выручка сосредоточена в одной стране:
- **United Kingdom** — **9,003,097.96**

Остальные страны сильно уступают по объёму продаж:
- Netherlands — **285,446.34**
- EIRE — **283,453.96**
- Germany — **228,867.14**
- France — **209,715.11**

## Основные выводы
- Бизнес имеет выраженную сезонность: максимальная выручка наблюдается осенью и в конце года.
- Основная часть выручки приходится на рынок **United Kingdom**.
- Существенная доля продаж формируется небольшим числом крупных клиентов.
- Средний чек достаточно высокий, что говорит о заметной стоимости одного заказа.
- SQL-анализ позволяет быстро получить ключевые бизнес-метрики и выявить базовые закономерности в продажах.

## Как запустить

Перед запуском убедитесь, что файл `data.csv` находится в папке `data/`.

### 1. Открыть PostgreSQL

В терминале выполните:

```bash
"/Applications/Postgres.app/Contents/Versions/18/bin/psql" -p5432 -U postgres eda_sql_project_db
```
### 2. Выполнить SQL-скрипты

